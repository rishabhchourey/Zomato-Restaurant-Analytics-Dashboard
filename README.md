# 🍽️ Zomato Restaurant Analytics Dashboard (Power BI)

## 📌 Project Overview
This is an end-to-end data analytics project built using Power BI on Zomato restaurant dataset. The project focuses on extracting meaningful insights from raw data and presenting them through an interactive dashboard.

---

## 🔄 Project Workflow
1. Data Cleaning (Excel)
2. Data Transformation
3. DAX Measures Creation
4. Data Visualization (Charts)
5. Slicers & Filters
6. Dashboard Designing

---

## 📊 Key Insights
- Monthly sales and order trends
- Top performing restaurants and cities
- Most popular food categories
- Customer ordering patterns

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel (Data Cleaning)
- Power BI (Data Visualization & DAX)

---

## 🚀 Project Highlights
- Built interactive and dynamic dashboard
- Used advanced DAX measures
- Applied filters and slicers for better user experience
- Designed visually appealing UI
